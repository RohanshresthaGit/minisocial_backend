# MiniSocial

A modern social platform API that connects people, shares moments, and builds communities.

## What is MiniSocial?

MiniSocial is a social networking platform that allows users to:
- **Create Accounts** - Register with email and get started
- **Authenticate Securely** - Login with encrypted JWT tokens
- **Connect with Others** - Build your social network
- **Share & Engage** - Post content and interact with the community

Think of it as a modern, lightweight social network built on a powerful backend infrastructure.

## Key Features

✨ **User Authentication**
- Quick and easy registration process
- Secure email-based login
- Never-expire with JWT token refresh technology

🔐 **Security First**
- Military-grade password encryption
- Secure token-based authentication
- Protected user data

⚡ **Fast & Reliable**
- Built on PostgreSQL for consistency
- RESTful API for easy integration
- Clean, predictable responses

📱 **Developer Friendly**
- Well-documented API endpoints
- Clear error messages
- Easy to integrate with mobile and web apps

## Getting Started

### For Users
1. Download or access the MiniSocial app
2. Click "Sign Up" and create your account
3. Login with your email and password
4. Start connecting with people!

### For Developers
Want to build on top of MiniSocial? Check out [DEVELOPMENT.md](DEVELOPMENT.md) for the complete setup guide.

## Tech Stack

- **Backend:** Django (Python web framework)
- **Database:** PostgreSQL (reliable data storage)
- **Security:** JWT authentication with BCrypt encryption
- **API:** REST API with Django REST Framework
